# discord-notion-bot

A Discord bot that saves messages to a Notion database.

## Render settings

- Service type: Background Worker
- Build command: `npm install`
- Start command: `npm start`

## Environment variables

Add these in Render:

- `DISCORD_TOKEN`
- `NOTION_TOKEN`
- `NOTION_DATABASE_ID`

## Discord usage

```text
!notion your memo
```
